Whaaaat is Polis

Sabe quando você explode desabafa no Facebook sobre aquele problema na sua rua pra todo mundo ouvir? Então... O Polis é um espaço colaborativo para você expor os problemas bugs que rolam pela cidade.
No Polis você e várias outras pessoas podem mostrar o que anda de errado e recebe atualizações sobre suas reclamações e atividades perto da sua casa!

Faça a diferença no lugar certo e da forma mais eficiente!

Apenas expor as suas reclamações e sugestões sobre a cidade em redes sociais pode não ser a melhor forma de resolver o problema.

A galera aqui do Polis sabe que expor suas ideias e reclamações em redes sociais é muito importante como forma de cidadania inclusive amamamos fazer isso, porém, nem todos os posts e perfis têm um alcance do tamanho que o problema exige para ser notado.

Mas olha só: juntos somos mais fortes!

  • Você não está alone sozinho. Una forças com outros cidadãos engajados.
  • Saiba onde exatamente estão acontecendo os outros problemas e o que está sendo feito a respeito.
  • Receba atualizações sobre as suas demandas, sobre o seu bairro e outras informações que você tem interesse.

O Polis não é apenas um local para você dizer o que não está certo, também é uma central de informações. Aqui você pode saber quando abriu um novo buraco na pista do seu bairro após uma chuva, obras que irão acontecer nas redondezas, quando a iluminação da esquina do seu trabalho receberá atenção, se existe foco do mosquisto da dengue próximo à sua casa e como os responsáveis estão tratando disso E MUITO MAIS!

Perguntas Frequentes

Ainda somos um prjeto jovem, então é natural que existam muitas dúvidas. Aqui embaixo elencamos as respostas para as perguntas que você pode estar se fazendo ;)
Se não encontrou o que buscava, nos envie uma mensagem perguntando o que quiser clicando aqui.

  • Eu preciso pagar alguma coisa para usar o Polis?

    Negativo. O Polis é totalmente gratuito e você pode usar todas as features de acordo com a nossa política de utilização.

  • Como eu faço para expor as necessidades da cidade?

    Primeiramente você deve criar um perfil. clique aqui para criar. A partir daí você fará parte da nossa comunidade galera. Depois disso você poderá acessar o seu perfil onde haverá todas as features disponíveis. Uma destas features chama-se "ocorrências", é aqui onde você poderá contribuir com as suas observações.
    Não se preocupe com as partes técnicas, é tudo muito intuitivo e você não ficará perdido! Mesmo assim lá haverá um guia que irá ajudá-lo com qualquer coisa que você quiser saber.

  • Quem pode utilizar o Polis?

    Todos podem utilizar o Polis dentro da nosso política de utilização que não permite principalmente perfis fakes (falsos).

  • Para que o Polis utiliza os dados das ocorrências?

    Os dados inseridos pelos usuários são utilizados para que o poder público tenha acesso às informações sobre o andamento da cidade. Através destas informações disponíveis a prefeitura poderá resolver os problemas expostos, criar medidas para previnir situações recorrentes, responder os cidadãos diretamente dentro da plataforma, ouvir as sugestões de melhorias, comunicar situações, e mais outras possibilidades que serão alcançadas de acordo com o que o povo expor.
    No perfil do usuário irá aparecer apenas o nome e informações pertinentes às conquistas do usuário, como quantidade de ocorrências, quantidades de respostas, etc. Os dados do usuário não serão incorporados na API. Clique aqui para ler mais sobre a API

  • O que é a API do Polis?

    API é arquitetura de computação que disponibiliza informações sobre determinado serviço disponível em uma plataforma/sistema web. O Polis foi desenvolvido utilizando o Django Web Framework e também implementa o Django Rest Framework para servir os recursos disponíveis no sistema. Através de chamadas aos endpoints da API você poderá utilizar algumas informações disponíveis nas ocorrências e utilizá-las em seu próprio sistema. Obs: a API da acesso apenas às informações básicas disponíveis no sistema e limita-se às ocorrências mais recentes, sem acesso às informações de usuários.

  • Como faço para ter acesso à API do Polis?

    Por enquanto a API Rest está aberta para todos os desenvolvedores utilizarem, basta acessar o link a seguir: Polis Rest API
